Поиск

Для поиска по базе данных перейдите на страницу "поиск".

  • Для фильтрации результатов по категориям нажмите на кнопку "метаданные", "тематика" или "геоданные". Выберите необходимые опции во всплывающем окне. Закройте всплывающее окно и нажмите на кнопку поиска.
  • Для поиска по тексту вводите слова в строку поиска, затем нажмите на кнопку поиска. При поиске по нескольким словам вводите их через один пробел.
